# Robot Command Simulator

A Java program that simulates a robot executing a series of commands using queues and stacks.

## What it does
Reads a list of robot commands from a text file and simulates the robot’s actions in order, supporting undo/redo functionality.

## Key Features
- Queue and stack data structures  
- Command simulation logic  
- File input/output for reading instructions 

## How to Run
Compile and run from the command line:

```bash
javac -d out src/*.java
java -cp out RobotSimulator
```

## Concepts Demonstrated
- Queues and stacks
- Simulation and algorithmic logic
- Object-oriented programming in Java
